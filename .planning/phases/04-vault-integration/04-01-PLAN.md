```python
def fetch_todays_signals(client: WeaviateClient) -> list[dict]:
    signals_col = client.collections.get("Signals")
    response = signals_col.query.fetch_objects(
        filters=(
            Filter.by_property("tier").contains_any(["BRIEF", "VAULT"])
            & Filter.by_property("published_date").greater_or_equal(yesterday.isoformat())
        ),
        limit=100,
    )
    return [
        {"uuid": str(obj.uuid), "title": obj.properties.get("title", ""), ...}
        for obj in response.objects
    ]
```

    **Translator module (src/pipeline/translator.py):**

    Create with these exact functions:

    1. `get_vault_seeds_path() -> Path` — reads `os.environ.get("OBSIDIAN_VAULT_PATH")`, raises `EnvironmentError` if not set, raises `FileNotFoundError` if `01_seeds/` missing, raises `PermissionError` if not writable. Returns `Path(vault_root) / "01_seeds"`.

    2. `fetch_vault_signals(client: WeaviateClient) -> list[dict]` — queries Signals collection with `Filter.by_property("tier").equal("VAULT")` and `Filter.by_property("published_date").greater_or_equal(yesterday_iso)`. Post-filters with `float(obj.properties.get("confidence") or 0.0) >= 0.8`. Returns dicts with keys: uuid, title, abstract, source_url, score, confidence, matched_pattern_ids, cluster_id. Use yesterday as cutoff (same as analyst.py pattern: `datetime.now(timezone.utc) - timedelta(days=1)`).

    3. `rank_and_cap(signals: list[dict], max_n: int = 3) -> list[dict]` — sorts by `float(s.get("confidence") or 0.0)` descending, returns `[:max_n]`.

    4. `_load_deposited_urls(seeds_path: Path) -> set[str]` — globs `seeds_path/*.md`, regex-parses frontmatter between first `---` pair (non-greedy `re.match(r'^---\n(.*?)\n---', text, re.DOTALL)`), extracts `arxiv_url:` value. Returns set of URLs. Swallows per-file exceptions.

    5. `slugify_title(title: str) -> str` — `re.sub(r'[^a-z0-9]+', '-', title.lower()).strip('-')`. Must not return empty string (fallback to "untitled").

    6. `render_seed_note(signal: dict, agent_summary: str = "") -> str` — produces full markdown with YAML frontmatter. Fields in order:
       - `folder: "01_seeds"`
       - `type: "seed"`
       - `created: {date.today().isoformat()}`
       - `status: "raw"`
       - `urgency: "low"`
       - `context: "work"`
       - `source: "meridian"`
       - `spark_stage: "capture"`
       - `tags: [seed, auto-deposit]`
       - `processed_date: ""`
       - `promoted_to: ""`
       - `agent_context: "Auto-deposited VAULT signal from Meridian — {agent_summary}"`
       - `auto_deposit: true`
       - `signal_uuid: "{signal['uuid']}"`
       - `confidence: {signal['confidence']}`
       - `score: {signal['score']}`
       - `matched_patterns: {signal['matched_pattern_ids']}`
       - `arxiv_url: "{signal['source_url']}"`

       Body: `# {title}\n\n## Abstract\n\n{abstract}\n\n## Matched Patterns\n\n{pattern_ids_as_bullet_list}\n\n## Cluster Context\n\nCluster: {cluster_id}\n`

    7. `write_translator_heartbeat(deposited: int) -> None` — writes `heartbeat/translator.json` with `last_run` (UTC ISO), `seeds_deposited`, `status: "ok"`. Same pattern as `write_briefing_heartbeat`.

    8. `TRANSLATOR_HEARTBEAT_PATH = Path("heartbeat/translator.json")` — module-level constant.

    9. `run_translator_pipeline() -> None` — orchestrator:
       - Call `get_vault_seeds_path()` OUTSIDE the try/except (fail loud on misconfiguration)
       - `client = get_client()`
       - Inside try/except (swallow all runtime errors):
         - `signals = fetch_vault_signals(client)`
         - `candidates = rank_and_cap(signals, max_n=3)`
         - `deposited_urls = _load_deposited_urls(seeds_path)`
         - Loop over candidates, skip if `signal["source_url"] in deposited_urls`
         - Track `deposited` counter, break if `deposited >= 3`
         - Generate filename: `slugify_title(signal["title"]) + ".md"`, check `(seeds_path / filename).exists()` for collision — if exists, try `-2`, `-3` etc.
         - `(seeds_path / filename).write_text(render_seed_note(signal))`
         - Increment deposited counter
         - After loop: `write_translator_heartbeat(deposited)`
       - except Exception as e: `logger.error(f"Translator pipeline failed: {e}")` — do NOT re-raise
       - finally: `client.close()`

  ```python
  HEARTBEAT_PATH = Path("heartbeat/scout.json")

  def write_heartbeat(paper_count: int, scored_count: int) -> None:
      HEARTBEAT_PATH.parent.mkdir(parents=True, exist_ok=True)
      HEARTBEAT_PATH.write_text(json.dumps({
          "last_run": datetime.now(timezone.utc).isoformat(),
          "papers_fetched": paper_count,
          "papers_scored": scored_count,
          "status": "ok",
      }))
  ```

  `run_scout_pipeline()` — follow the exact skeleton from RESEARCH.md "Full Pipeline Skeleton":
  1. `target_date = date.today() - timedelta(days=1)`
  2. `client = get_client()`
  3. try/finally with `client.close()` in finally
  4. Inside try: fetch_arxiv_papers -> fetch_pattern_keywords -> keyword_filter(papers, keywords, cap=50)
  5. Loop over filtered papers: normalize_arxiv_id, check arxiv_id_exists (skip if True), get_top_patterns, score_paper (try/except per paper — log and continue on failure), assign_tier, write_signal with full data dict including reasoning
  6. After loop: write_heartbeat(len(papers), scored_count)
  7. Log summary with logger.info
